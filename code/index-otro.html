Rankings y Desempeño

¿Cómo se calcula la Puntuación General?

La puntuación general de cada estudiante (de 0 a 100%) se calcula ponderando diferentes aspectos de su desempeño:

  • Promedio Final de Calificaciones: %
  • Porcentaje de Entregas de Actividades: %
  • Calificación del Proyecto Final: %
  • Calificación de la Bitácora: %
  • Porcentaje de Asistencia: %

Cada uno de estos componentes se normaliza (escala de 0 a 1) antes de aplicar su peso correspondiente. La puntuación de equipos y grupos se basa en el promedio de las puntuaciones individuales de sus miembros, con una posible bonificación por equidad en el desempeño.
